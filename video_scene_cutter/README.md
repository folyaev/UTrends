# Video Scene Cutter

Отдельная экспериментальная утилита для вырезания повторяющихся визуальных сцен из видео.

Идея: загрузить ролик блогера, просканировать кадры по цветовой HSV-гистограмме, найти часто повторяющиеся визуальные сцены и дать выбрать, что удалить или оставить. Это удобно для роликов, где говорящая голова снята в одной локации, а перебивки/вставки визуально отличаются.

## Как пользоваться

1. Выбери входное видео.
2. Нажми `Analyze`.
3. В списке появятся самые повторяющиеся сцены.
4. Выбери сцену: обычно talking-head локация будет самой длинной.
5. Выбери режим:
   - `Remove selected scene` - удалить выбранную повторяющуюся сцену.
   - `Keep only selected scene` - оставить только ее.
6. Выбери путь сохранения и нажми `Export`.

Параметры:

- `Sample step` - как часто брать кадр для анализа.
- `Similarity` - насколько похожими должны быть кадры для одной сцены.
- `Merge gap` - склеивать близкие куски одной сцены.
- `Min segment` - выкидывать слишком короткие итоговые куски.
- `ROI` - можно ограничить анализ областью кадра, если фон/лицо находятся в конкретной зоне.
